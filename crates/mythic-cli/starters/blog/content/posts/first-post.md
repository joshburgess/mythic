# Getting Started with Mythic

Welcome to your new blog! This is your first post.

## Writing Content

Create markdown files in the `content/` directory. Each file becomes a page.

## Frontmatter

Use YAML frontmatter to set metadata:

```yaml
---
title: "My Post"
date: "2024-01-15"
tags:
  - example
---
```

## Code Highlighting

Syntax highlighting works out of the box:

```rust
fn main() {
    println!("Hello from Mythic!");
}
```

## Next Steps

- Edit this post or create new ones in `content/posts/`
- Customize templates in `templates/`
- Run `mythic serve` for live preview
