# mythic-core

Core library for the [Mythic](https://github.com/joshburgess/mythic) static site generator.

Provides config loading, content discovery, the incremental build pipeline, caching, taxonomies, feeds, sitemap generation, data loading, i18n, plugin system (Rust hooks + Rhai scripting), content linting, accessibility auditing, and link checking.

This crate is primarily intended to be used as a dependency of other Mythic crates. For the CLI tool, install [`mythic-cli`](https://crates.io/crates/mythic-cli).
