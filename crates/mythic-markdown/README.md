# mythic-markdown

Markdown processing for the [Mythic](https://github.com/joshburgess/mythic) static site generator.

Provides YAML/TOML frontmatter parsing, markdown-to-HTML rendering via pulldown-cmark, syntax highlighting via syntect, shortcode processing, table of contents extraction, admonitions, and math rendering support.

This crate is primarily intended to be used as a dependency of other Mythic crates. For the CLI tool, install [`mythic-cli`](https://crates.io/crates/mythic-cli).
