# mythic-assets

Asset processing for the [Mythic](https://github.com/joshburgess/mythic) static site generator.

Provides image optimization with automatic WebP generation and responsive `<picture>` tags, CSS/JS bundling and minification with content-hashed filenames, Sass/SCSS compilation, and SRI integrity hash generation.

This crate is primarily intended to be used as a dependency of other Mythic crates. For the CLI tool, install [`mythic-cli`](https://crates.io/crates/mythic-cli).
